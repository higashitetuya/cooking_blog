## サイト概要
自分や他人が投稿した料理を閲覧・コメント・シェア等が出来るコミュニティサイト

## ターゲットユーザー
家で料理を作るユーザー

## 主な利用シーン
投稿された料理を閲覧しながら料理ができたりコメントができる

## ターゲットユーザーの利用目的
他の人の料理を知ることができる
コメントやいいねを行うことによりコミュニティを広げることができる
